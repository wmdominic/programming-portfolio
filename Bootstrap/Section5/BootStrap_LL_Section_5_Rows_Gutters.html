A New Row, One Less Column

This is all within the same div container, though it is a new row. Because it's within the same overall container, it continues to set up the rows in the same grid.

But wait... What about the space?!

This row only has two columns, whereas the previous one had three. As you can see, it does not matter if one row has more or less columns as another. The columns will evenly split the space and will take up the width of the overall container.
