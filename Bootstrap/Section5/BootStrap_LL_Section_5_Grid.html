To read the table above: When using the specific container class, the container will take up 100% of the viewport when it hits a size of 576 pixels or less (regardless of size).

Once the specified container class hits its particular breakpoint or px amount, it will remain at 100% of the container.

For example, .container-md will remain its designated size until the viewport reaches 768px (where the container will be a width of 720px). Once it reaches this point, it will remain at 100% width of the viewport.
